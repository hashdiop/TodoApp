# Installation de Tailwind CSS

## Étapes d'installation

1. **Installer les dépendances** :
```bash
cd frontend
npm install
```

Cela installera automatiquement :
- `tailwindcss`
- `autoprefixer`
- `postcss`
- `react-icons`

2. **Vérifier la configuration** :
Les fichiers de configuration sont déjà créés :
- `tailwind.config.js`
- `postcss.config.js`

3. **Démarrer l'application** :
```bash
npm start
```

## Icônes utilisées

L'application utilise `react-icons` avec le pack `Fi` (Feather Icons) :
- `FiLogIn` - Connexion
- `FiUserPlus` - Inscription
- `FiMail` - Email
- `FiLock` - Mot de passe
- `FiLogOut` - Déconnexion
- `FiPlus` - Ajouter
- `FiEdit2` - Modifier
- `FiTrash2` - Supprimer
- `FiCheck` - Valider
- `FiX` - Annuler
- `FiCheckCircle` - Tâche complétée
- `FiCircle` - Tâche non complétée
- `FiLoader` - Chargement
- `FiSend` - Envoyer
- `FiArrowLeft` - Retour
